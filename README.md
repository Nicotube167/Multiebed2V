# Multiebed2V
✅ Unlimited named beds per player. * ✅ After waking up in a new bed, a GUI opens. * ✅ Click Name Bed. * ✅ Type the bed’s name in chat. * ✅ Every bed is saved. * ✅ When you die and click Respawn, a GUI opens with all your beds. * ✅ Click a bed to respawn there. * ✅ Broken beds are automatically removed. * ✅ Configurable maximum number of beds. * ✅
